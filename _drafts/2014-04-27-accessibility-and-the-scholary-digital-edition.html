Accessibility = universal design: Values Followthrough

As much as possible, design for all readers; build on existing work on usability and accessibility to create better DH-specific accessibility tutorials and standards. Library website accessibility and usability work is a fruitful place for research, as the structure and use of library websites is similar to that of a complex digital scholarly edition.
